## Login Instructions

### First Time Login (Password Setup)

1. **Go to**: http://127.0.0.1:8080 or http://localhost:8080
2. **Select Role**: Choose "Faculty"
3. **Enter Email**: Use your assigned email from the table above (e.g., 2000001@geu.ac.in)
4. **Enter Password**: Type ANY password you want to set
5. **Click Login**: Your password will be saved permanently
6. **Important**: Remember this password! It cannot be changed later.

**Success Message**: You will see "Password set successfully! Please remember this password for future logins."

### Subsequent Logins

1. **Select Role**: Choose "Faculty"
2. **Enter Email**: Use the same email you used during first login
3. **Enter Password**: Use the exact password you set during first login
4. **Click Login**: You will be logged in successfully
